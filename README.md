# Phonebook

A simple web application to keep track of your contacts, built with React and
Node.js. This is a part of the exercise from FullstackOpen.

🌍 **Live Site:** [https://phonebook-mtdi.onrender.com](https://phonebook-mtdi.onrender.com)

## What it does
- Add new names and numbers to your phonebook.
- Update the number of an existing contact.
- Delete contacts you no longer need.
- Search and filter through your saved list.

## Under the hood
- **Frontend:** React
- **Backend:** Express:
- **Data:** In-memory array (resets on server restart)
